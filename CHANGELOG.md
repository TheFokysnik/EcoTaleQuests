# EcoTaleQuests — Changelog

## v1.4.0 — 2026-02-21

### Новое
- **Настройка символа валюты** — новая опция `CurrencySymbol` в секции General конфига. По умолчанию `$`, можно поставить любой символ (`€`, `G`, `¥` и т.д.).
- **Округление валюты** — новая опция `RoundCurrency` в секции General. При `true` все суммы округляются вверх до целого числа (`Math.ceil`).
- **Admin GUI: новые настройки** — в секцию General админ-панели добавлены:
  - S15: Currency Symbol (текстовое поле + кнопка «Применить»)
  - S16: Round Currency (переключатель ON/OFF)
- **`MessageUtil.formatCoins(double, boolean)`** — перегрузка с параметром округления.

### Изменения
- Панель квестов (QuestPanel.ui) перекрашена в тёплую золотисто-коричневую палитру SkillTree.
- Панель админа (AdminPanel.ui) перекрашена в ту же палитру.
- Все 3 места с хардкод `"$"` в PlayerQuestsGui заменены на `formatRewardCoins()` с чтением конфига.
- Иконка категории `EARN_COINS` теперь берёт символ из конфига.

## v1.4.0 — 2026-02-17

### Новое
- **Полная абстракция провайдеров экономики и уровней** — `EconomyProvider` / `LevelProvider` интерфейсы с Bridge-паттерном.
- Поддержка провайдеров: `ecotale`, `economyapi`, `generic` (экономика); `rpgleveling`, `endlessleveling`, `mmoskilltree`, `generic` (уровни).
- **Generic-провайдеры** — конфигурируемые reflection-адаптеры для любого стороннего плагина.
- Нативная GUI-панель квестов (Daily / Weekly / Active / Top вкладки).
- Admin GUI с полным редактором настроек, квестов, типов.
- Система рангов (E → D → C → B → A → S), Rank Points, прогрессия.
- Таймеры квестов с авто-провалом.
- Quest Board (NPC-интерактор).
- Локализация: 8 языков (en, ru, de, fr, es, pt_br, cs, hu).
- HUD-нотификации через `NotificationUtil`.

### Исправления
- Баг с начислением XP при завершении квеста.
